# 💇 HairBook

Sistema de gestión de citas para una peluquería donde los clientes pueden agendar citas en línea.

---

## 📌 Descripción

HairBook es una plataforma que permite a los clientes reservar citas en una peluquería de manera digital, mientras que el administrador puede gestionar las citas y horarios disponibles.

---

## ⚙️ Funcionalidades principales

- Registro de usuarios
- Inicio de sesión
- Visualización de horarios disponibles
- Agendamiento de citas
- Panel de administración
